# HR-Flow AI: Modern SaaS Platformu

Bu proje, geleneksel HR süreçlerini Yapay Zeka (NLP & Semantik Analiz) ile dijitalleştiren, FastAPI backend ve React frontend mimarisine sahip profesyonel bir SaaS platformudur.

## 🚀 Öne Çıkan Özellikler

- **Hibrit Skorlama:** Semantik benzerlik (Sentence Transformers) ve teknik anahtar kelime eşleştirme ile %100 isabetli aday sıralama.
- **Premium UI:** Indigo-500 tabanlı, Dark Mode destekli, Lucide ikonları ve modern geçiş efektleri içeren kullanıcı arayüzü.
- **Detaylı Analiz (Deep Dive):** Radar grafikler, aday özgeçmişinden kanıtlar ve otomatik uzmanlık etiketleri.
- **İletişim Yönetimi:** CV'lerden otomatik ayıklanan Mail, LinkedIn, GitHub linkleri ve tek tıkla kopyalama özelliği.
- **Akıllı Filtreleme:** İK terimlerini ve gereksiz dolgu kelimeleri (fluff) eleyerek sadece gerçek yetkinliklere odaklanan analiz motoru.

## 🛠️ Kurulum ve Çalıştırma

### 1. Backend (FastAPI)
```bash
# Bağımlılıkları yükleyin
pip install -r requirements.txt

# Sunucuyu başlatın
python backend/main.py
```

### 2. Frontend (React + Vite)
```bash
cd frontend
npm install
npm run dev
```

Platforma `http://localhost:5173` (veya Vite'ın atadığı port) üzerinden erişebilirsiniz.
